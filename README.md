# Shiane Carl Roblas — Portfolio Resume

Built with React + Vite + Tailwind CSS

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Run dev server
npm run dev

# 3. Build for production
npm run build
```

## How to Customize

- **Your info**: Edit `src/data.js` — name, bio, projects, skills, etc.
- **Profile photo**: Add your photo as `src/assets/profile.jpg`
- **Certificates**: Upload images directly in the browser, or import them in `src/components/Certificates.jsx`
- **Resume PDF**: Place your resume PDF in `public/resume.pdf` so the Download button works

## Deploy to Vercel

1. Push this project to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Framework: **Vite** (auto-detected)
5. Click **Deploy** — done! 🚀
